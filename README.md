# 🌀 Naruto Hand Tracking App

A simple real-time hand tracking web app that triggers **Naruto** and **Sasuke** powers using your hands.

---

## 🚀 How to Use
1. Open `index.html` in your browser  
2. Allow camera access when prompted  
3. Start using hand gestures to activate powers  

---

## 🎮 Controls

- ✋ **Right Hand Open** → Activates **Naruto Power**
- ✋ **Left Hand Open** → Activates **Sasuke Power**
- 🔵 **Blue Lines** → Visual hand tracking (helps confirm detection is working)

---

## ⚙️ Features
- Real-time hand tracking using webcam  
- Lightweight and browser-based (no installation required)  
- Simple gesture-based interaction  
- Works on most modern devices  

---

## ⚠️ Note
- Glow effects were removed to ensure **better compatibility and performance across all devices**
